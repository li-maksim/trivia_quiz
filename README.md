# Quiz App

Квиз со случайными вопросами. Посмотреть можно по [ссылке](https://li-maksim-quiz.netlify.app/).
Вопросы загружаются из [QuizAPI](https://quizapi.io/).

## Функции

- Загрузка случайных вопросов из API
- Проверка правильности ответов
- Подсчет очков пользователя
- Ограничение по времени: на все вопросы нужно ответить за 5 минут
- Светла/тёмная тема

## Технологии

- React
- TypeScript
- Vitest и React testing library
- Tailwind

## Структура проекта

- components: React-компоненты (чекбоксы, форма, таймер, кнопки и т.д.)
- tests: тесты основных компонентов (работа чекбоксов, формы, таймера)
- utils: вспомогательные файлы (хук для фетчинга данных, интерфейсы TS, данные для тестов)
